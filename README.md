# y99-api
A set of templates for interacting with api2.y99.in especially with messages. This is useful for things like moderation and bots.

When I first started looking into the network requests I found the API agent which I used deobfuscation programs to generate "Deobfuscation Attempt.js". Through tinkering in the network tab I eventually got to the conclusions found in "Tempaltes.txt", as time goes on I will continue to update this repo with more up-to-date info because some of the conclusions were incorrect and I will give each one its own .js file.
